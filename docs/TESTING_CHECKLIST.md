## 16. Edge Cases

### Невалидный адрес:
- [ ] Ввести `abc123` — перенаправление на 404 или сообщение об ошибке
- [ ] Ввести `0x0000000000000000000000000000000000000000` — graceful handling

### Контракт без транзакций:
- [ ] Плашка "No transaction data available"
- [ ] Ссылка "Try another contract"

### Неверифицированный контракт:
- [ ] Дашборд загружается (без ABI данных для function names)
- [ ] Функции показаны как hex selectors

### Сеть/API:
- [ ] Отключить интернет → сообщение об ошибке
- [ ] Медленный интернет → спиннер виден, таймаут 15 секунд
- [ ] Обновить страницу во время загрузки → без краша

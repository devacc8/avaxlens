## Workshop Insights (Web3 Enterprise by Raquel/Tether)

### Key Enterprise Mindset
> **"Their biggest fear is being the person who approved something that went wrong"**

Enterprise users care about **risk mitigation**, not innovation.

### Real Blockers for Enterprise

| Blocker | Why It's a Problem |
|---------|-------------------|
| **No rollback** | Can't undo a transaction |
| **No chargebacks** | No fraud protection |
| **If you lose keys** | Lost keys = lost funds forever |
| **No customer support** | Who to call when something breaks? |
| **Legal/compliance** | "Is this legal? Can we get audited?" |

### Framework: 4 Questions for Enterprise

| Question | AvaxLens Answer |
|----------|-----------------|
| **What's the behavior change?** | Just paste an address — minimal |
| **What's the worst case?** | Data is wrong — just look again, no financial loss |
| **Who do they call?** | Documentation + support email |
| **Is it better?** | Faster than Explorer, free vs Tenderly ($49) + Dune ($39) |

### Why AvaxLens Fits Enterprise

✅ **No financial risk** — just viewing analytics
✅ **Zero behavior change** — paste address, get results
✅ **Transparent** — data from public APIs
✅ **Auditable** — can verify the data

## Q&A Prep

### "Why not just use Tenderly?"
> "Tenderly costs $49/month for Team plan (100k requests limit), requires project setup, and is multi-chain — not optimized for Avalanche. We're free, instant, and Avalanche-native."

### "Why not Dune?"
> "Dune costs $39/month for basic access, requires SQL knowledge, and limits you to 500 queries/mo on Team plan. We're visual, instant, and require zero query writing."

### "How do you make money?"
> "Free tier always available. Pro tier for teams that need alerts and history."

### "What if APIs shut down?"
> "Routescan and Snowtrace are ecosystem tools. Fallback: direct RPC parsing."

### "Why Avalanche?"
> "Growing ecosystem, no native tools, developer-focused community."

### "What if the data is wrong?"
> "We pull directly from Routescan and Snowtrace APIs — the same data Avalanche Explorer uses. Users can verify themselves."

### "Is this safe for enterprise use?"
> "AvaxLens is a read-only analytics tool. No transactions, no keys, no financial risk. We're just making the existing blockchain data easier to understand."

### "What about compliance?"
> "All data comes from public Avalanche APIs. No private data, no KYC. Fully transparent and auditable."

### "Who do I call if something breaks?"
> "We provide documentation and support email. For enterprise, we can offer dedicated support in the Pro tier."
